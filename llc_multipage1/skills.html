Skills

So, I'm fairly new to coding and at times I feel like I need to learn a new language just to understand what experienced coders are saying. HTML. CSS. Back-end. C. C++. Front-end. JavaScript. Ruby. Git. PHP. Bug. Debug. Uhm... say WHAT now??? Seriously, at times it sounds like Klingon to me. Scotty, beam me out of here!

Fortunately, the more I practice writing code, the more I start to feel like I don't need that sweet Scotty anymore. I by no means know everything and I'm still CMD+C and CMD+V quite a few error messages in everybody's best friend, Google.But I do feel a lot more comfortable working with:

  • HTML
  • CSS

This website about me you're so generously spending some of your precious time on? Coded this all by myself! With a bit of help from a tutorial, yes. But still impressive for a beginner, right?

And I've started to experiment a little bit with:

  • JavaScript
  • Ruby

I will probably keep dabbling with these 4 languages for the coming few months. But I'm looking forward to adding Python and PHP to the mix as well, since I hear a lot of coders mentioning them. Might as well get into this actio as well.

Now, if only I could only find a 'How To Speak Like A Coder' cheatsheet in time for my next Meetup. Would be nice to be do some rubberducking with some of the other coders. Ha! I already know that one.
